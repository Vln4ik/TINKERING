Папка для дополнительных корневых/промежуточных сертификатов (.crt), которые нужно добавить в доверенные CA внутри контейнера `api`.

Использование:
- положите сюда один или несколько файлов `*.crt`
- пересоберите/перезапустите контейнер (`docker compose up --build -d`)

Контейнер при старте копирует `certs/*.crt` в `/usr/local/share/ca-certificates/` и выполняет `update-ca-certificates`.
